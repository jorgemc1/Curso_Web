A Tag <div>

A tag <div> é usada para dividir e organizar partes de uma página HTML.
Ela funciona como um contêiner genérico para agrupar elementos.
É muito utilizada para estruturar layouts e aplicar estilos com CSS.
Também ajuda na manipulação de elementos usando JavaScript.
