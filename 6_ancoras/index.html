Página 2

no HTML a tag <a> é conhecida como âncora (“anchor” em inglês).

<a href="https://google.com">Google</a>

Nesse caso:
<a> = tag âncora
href = atributo da âncora
o texto “Google” = conteúdo do link

Importante:
A âncora é a tag <a>
href não é a âncora; ele é um atributo da tag
